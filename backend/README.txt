# INFORMATIONS GENERALES

## Titre du projet : Système de mise en vente d'articles "e-commerce"
 

# INFORMATIONS METHODOLOGIQUES

##Spécifications : 
Il est important de notifier que pour ce projet, il n'y a pas de fronttend donc tous les testes se feront
avec l'outil Postman

Le projet permet d'ajouter des utilisateurs, de leur assigner des rôles et de pouvoir ajouter des articles 'stuffs' pour chauque vendeur.


## Description des sources et méthodes utilisées pour collecter et générer les données :
Pour le développement, les environnements techniques sont : 
Editeur de code : VSCode
Runtime         : Node
Framework       : Express
Base de donnée  : MongoDB
